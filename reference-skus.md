# Wawasan finished-goods SKU reference

Source: client **MONTHLY STOCKLIST 2026** — `STK` (finished goods) + `RS`
(ready-stock aroma candles) sheets. These are the sellable products that
appear as **order line items** in the OMS (`order_items.sku` / `.name`).

SQL Account remains the source of truth for pricing and the invoice itself;
the OMS only tracks the operational pipeline, so no prices are listed here.
Item entry in the app is free-text — use these codes/names verbatim so the
board, floor display and reports stay consistent across orders.

> Note: a few items are stocked in one UOM but **sold in cartons** on the
> invoice (e.g. STK008 Flying Horse — stocked PACK, billed `72 BOX/CTN`).
> Match the invoice's description + UOM on the order line.
